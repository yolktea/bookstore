# Bookstore Website

Web page built using HTML, CSS and JavaScript. It contains static pages of an online Book store. \
The website consists the following pages:
* Home page 
* Registration and user Login
* User profile page
* Books catalog
* Shopping cart
* Payment By credit card
* Order confirmation.
